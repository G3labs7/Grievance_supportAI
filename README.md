## 📌 What It Does
An AI-powered constituency grievance intelligence platform that:
- Classifies citizen complaints by category, urgency and sentiment using Gemini API
- Displays real-time analytics on an MP Staff Dashboard
- Auto-generates weekly executive briefing reports for parliamentary staff

## 🛠️ Tech Stack
React · TypeScript · Vite · Node.js · Google Gemini API · Firebase · Tailwind CSS
